# Screen Prep

A browser-based coding interview prep tool. Covers Python algorithms, SQL, statistics, ML concepts, and common traps.

## Usage

Open `index.html` in a browser or serve via any static file host.

```
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Features

- Python and SQL problem sets with in-browser execution
- MCQ quizzes and mock assessments
- Brainrot Cram flashcard deck with shortform reel mode
- Cheatsheet reference
